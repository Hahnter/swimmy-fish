# Swimmy Fish v36

## Short Summary

**Swimmy Fish** is a Pokémon-themed underwater endless runner inspired by Flappy Bird.

You play as **Magikarp**, swimming through an endless ocean while avoiding:

- Coral reefs
- Staryu
- Qwilfish
- Tentacool

Press **SPACE**, click, or tap to make Magikarp swim upward while gravity pulls it down. Survive as long as possible and beat your high score.

Designed to work as:
- Local HTML game
- Cloudflare Pages deployment
- Progressive Web App (PWA)
- Desktop and mobile browsers

# README

## Overview

Swimmy Fish is an endless survival game where the player controls Magikarp through an underwater obstacle course.

The game tracks:
- Survival Time
- Obstacles Cleared
- Best Score

The objective is simple:

**Don't hit anything.**

## Controls

### Desktop
- SPACE = Swim Up
- R = Restart
- H = Toggle Hitboxes

### Mobile
- Tap Screen = Swim Up
- Tap After Game Over = Restart

## Obstacles

- Coral
- Staryu
- Qwilfish
- Tentacool

Touching any obstacle results in game over.
